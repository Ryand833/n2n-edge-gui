Prerequisites:
--------------

1) This installer is built using NSIS, which is available at http://nsis.sourceforge.net/

2) The following additional NSIS plugins are required:

  a) SimpleSC - NSIS Simple Service Plugin
     http://nsis.sourceforge.net/NSIS_Simple_Service_Plugin

3) The n2n 'edge.exe' executable file is not provided with this source package.
